# MomentX
一款小天才好友圈增强Xposed模块，具有强大功能（存疑？
前身为`好友圈Ultimate-Xposed扩展`（为[好友圈Ultimate](https://github.com/sldjadkd/MomentUltimate)制作的模块）
## 主要功能
- 云控检测
- HTTPS请求/响应明文查看
- 主页增强
- 长按动态Dialog增强
- 发送自定义心情/动态 *（虽然受限严重）*
- 集成DeepSeek API
- 动态Dex加载
- 加密通信 *（用于躲避审核）*
- 可顺便检查好友圈更新
## 设备要求
- Android 7.0及以上
- Xposed API 82及以上
## 关于开源
本项目不开源，除非本人决定**永久停止维护**才会公开源代码。
